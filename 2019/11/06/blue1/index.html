cannon开发板STM32cube设置

Author: John Doe Date: 十一月 6, 2019  9:28:44

PS: 本设置只针对cannon开发板

首先打开STM32cube

没有下载STM32cube可以进行下载

stm32cube版本号要高于5.3否则也无法实现功能

cannon开发板的型号是STM32F401RET 需要在软件内先下载或者更新

还要下载名为X-CUBE-BLE1和X-CUBE-MEMS1的两个插件包

插件包都下载完成之后就可以开始进行配置了

①file-new project 创建新工程

②弹出来的页面里我们搜索STM32F401RET 找到之后双击它 这个时候我们就进入了工程的设置页面

③SYS设置修改

在左边栏里点击system core他会展开
展开之后我们点击SYS
这是展开了SYS的设置页面
我们把Debug这栏选择Serial
这样SYS就设置好了

④SPI1的设置修改

在左边栏点击connectivity展开
点击SPI1进入设置页面
Mode部分我们选择Full-Duplex Master
下方的Parameter Settings里把prescaler改成4
这样SPI1部分就设置好了

⑤USART1设置修改

在左边栏点击connectivity展开
点击USART1进入设置页面
Mode部分我们选择Asynchronous
这样USART1部分就设置好了

⑥点击additional software 进入设置页面

找到后缀为X -CUBE-BLE1的设置栏
按左边的下拉按钮 会有几个设置
application这栏右边的选择框选择beacon
HCU_TL这栏右边的选择框选择basic
HCU_TL_INTERFACE这栏右边的选择框选择UserBoard
controller和Utils这两栏要勾选上
然后点OK 保存 这样这一步我们就设置好了
这时左边栏会多出来刚刚设置的BLUE点击它
将MODE那栏里的两个都勾选上
在Platform Settings里 这时只有两个是对应上的
我们需要在芯片上自己设置一下
最右边的芯片部分 找到PC4
鼠标左键点一下PC4选择GPIO_EXTI4
然后去blue设置栏把Exti Line那一栏最右边的选择框选择PC4
相同的方法 在芯片那边找到PA4和PB1选择GPIO out
其中CSline选择PA4  resetline选择PB1

⑦NVIC的设置

在左边栏里点击system core他会展开
展开之后我们点击NVIC
这是展开了NVIC的设置页面
我们把写着EXTI line4 interrupt的那一栏勾选上
这样NVIC就设置好了

⑧GPIO的设置

在芯片部分最上面有两栏 pinout view 和 system view
点击system view
点击GPIO 这时就进入了GPIO的设置界面
在GPIO设置界面找到NVIC那栏点击它
我们把写着EXTI line4 interrupt的那一栏勾选上
这样GPIO就设置好了

⑨时钟设置

点击上方的Clock Configuration进入时钟配置
找到写着HCLK(MHZ)那栏 把数字改成72

⑩工程设置

点击上方的project manager
在这里面可以自己给工程起名字和选择路径以及一些其他的配置
工程路径和名字最好不要有中文 有可能会报错

over.
